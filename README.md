# 🚨 LifeLink v3.0

LifeLink is a web-based Emergency Response Platform designed to provide quick access to emergency assistance through AI-powered incident classification, GPS tracking, and emergency service discovery.

## Overview

LifeLink helps users report emergencies, identify incident types, view nearby emergency services, and access safety tools through an interactive dashboard.

The platform is built using pure HTML, CSS, and JavaScript and runs entirely in the browser.

---

## Features

### 🧠 AI Emergency Classification
- Classifies emergency descriptions using NLP-based pattern matching.
- Detects incidents such as:
  - Fire Emergencies
  - Medical Emergencies
  - Cardiac Emergencies
  - Road Accidents
  - Floods
  - Earthquakes
  - Security Threats
  - Physical Attacks
  - Women's Safety Alerts
  - Child Safety Alerts
  - General SOS Requests

### 📍 Live GPS Tracking
- Uses browser geolocation.
- Tracks user location in real time.
- Displays location accuracy information.

### 🗺️ Interactive Maps
- Built with Leaflet.js.
- Displays user location.
- Shows nearby emergency services.

### 🚑 Nearby Emergency Services
- Finds nearby:
  - Hospitals
  - Police Stations
  - Fire Stations
  - Clinics

### 🆘 SOS System
- Hold-to-activate SOS mechanism.
- Prevents accidental emergency activation.
- Generates emergency incidents for rapid response simulation.

### ⚡ Command Center
- Centralized emergency monitoring interface.
- Displays incident information and emergency status.

### 🛡️ Safety Modes
- Women's Safety Mode
- Child Safety Alerts
- General Emergency Assistance

### 📋 Emergency History
- Displays previously recorded emergency incidents.
- Shows:
  - Incident ID
  - Category
  - Severity
  - Status
  - Location
  - Time
